Notable design decisions:

  • Left menu, header, and footer share the same markup for each page
  • Dynamic Graph code can handle different bar values simply by changing constants.
  • Dynamic Graph code does not need to be changed if graph height changes.
  • If graph width changes, you only need to change 2 constants in the Dynamic Graph code.
  • Intuitive UX: after entering a number on the dynamic graph page, the textbox is cleared and refocused
